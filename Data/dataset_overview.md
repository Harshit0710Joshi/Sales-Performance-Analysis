Dataset Name: Sample Superstore  
Source: Public retail dataset  
File Format: CSV  
Number of Records: 9,994  
Type: Transaction-level sales data  

This dataset represents a global retail company and contains order-level information related to sales, profit, region, and product categories. It is suitable for analyzing regional and category-wise sales performance.

Suitability Assessment:
The dataset contains key attributes required for sales performance analysis, including region, product category, sales value, and profit. Although order-level dates are not present, the dataset is still effective for regional and category-wise performance evaluation.

Initial Data Observations:
• Sales values are positive across records  
• Profit includes both gains and losses  
• No missing values in Region, Category, Sales, or Profit columns  
• Dataset is clean enough for analytical processing

Column to Business Mapping:
Region → Used for regional performance comparison  
Category → Used for category-wise sales contribution  
Sales → Used to calculate revenue metrics  
Profit → Used to evaluate profitability  
Quantity → Used to analyze sales volume
