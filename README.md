# Sales-Performance-Analysis (End-to-End Data Analyst Project)
End-to-end Sales Performance Analysis using PostgreSQL, Python, and Power BI on real-world retail data to uncover regional and category-wise business insights

## Project Overview
This project analyzes sales performance of a global retail company using real-world transactional data. The objective is to evaluate regional and category-wise sales, identify profitability issues, and provide actionable business insights.

## Business Objectives
- Analyze sales and profit across regions
- Identify top and underperforming product categories
- Evaluate profitability and efficiency
- Support data-driven business decisions

## Dataset
- Source: Public Superstore retail dataset
- Records: ~9,994
- Type: Transaction-level sales data

## Tools & Technologies
- SQL: PostgreSQL
- Python: Pandas, Matplotlib, Seaborn
- Visualization: Power BI
- IDE: VS Code

## Workflow
1. Business problem understanding
2. Dataset exploration and documentation
3. Data cleaning and SQL setup
4. SQL-based business analysis
5. Python EDA and visualization
6. Interactive Power BI dashboard

## Key Insights
- West region generates the highest sales but shows margin inefficiencies
- Technology category contributes the most profit
- Furniture category underperforms across multiple regions
- High sales volume does not always result in high profitability

## Dashboard
The Power BI dashboard provides KPI tracking, regional and category-wise analysis, and interactive filters for decision-makers.

## Conclusion
This project demonstrates end-to-end data analysis skills, combining SQL, Python, and visualization to deliver business-ready insights.
